# ReactJS Boilerplate

ReactJS Boilerplate

## Getting Started

```
Please use an npm version that is >= 3.0.0 and < 5.0.0
Much better install node.js version 5.11.1
ReactJS version 16.4.1
```

### Installing

```
git clone https://github.com/Shankhadeep1234/React-BoilerPlace_without_CRA
npm install
npm start
```
